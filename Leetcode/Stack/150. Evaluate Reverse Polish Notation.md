### Optimal Solution

Reverse polish notation is simple, just take in all the numbers and when you encounter an operator simply pull the first two numbers and perform the operation and push it back in. 

```java
class Solution {
    public int evalRPN(String[] tokens) {
        Stack<Integer> stack = new Stack<>();
        for(String token : tokens){
            int eval = 0;
            if(token.equals("+")) eval = stack.pop() + stack.pop();

            else if(token.equals("-")) eval = (-1 * stack.pop()) + stack.pop();

            else if(token.equals("*")) eval = stack.pop() * stack.pop();

            else if(token.equals("/")) {
                int down = stack.pop();
                int top = stack.pop();
                eval = top / down;
            }

            else eval = Integer.parseInt(token);

            stack.push(eval);
        }

        return stack.pop();
    }
}
```
