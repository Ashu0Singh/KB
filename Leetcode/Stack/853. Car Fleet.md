### Optimal Solution

So here we have actually sorted the cars in order of their position on their way to target. And then we will start from the end and slowly move towards the beginning. We have to check at what time is a car reaching to the target and if the car on top is reaching after the current car then that becomes a car fleet. We have to do it for all the cars so that we can determine which of them are getting converted into a fleet and which are not. 
```java
class Solution {
    public int carFleet(int target, int[] position, int[] speed) {
        HashMap<Integer, Integer> map = new HashMap<>();
        Stack<Integer> stack = new Stack<>();

        for(int i = 0; i < position.length; i++){
            map.put(position[i], speed[i]);
        }

        List<Integer> list = new ArrayList<>(map.keySet());
        Collections.sort(list,Collections.reverseOrder());

        for(Integer pos : list){
            if(stack.isEmpty()) stack.push(pos);
            else {
                int carAhead = stack.peek();
                float carAheadTime = (float)(target - carAhead)/map.get(carAhead);
                float currCar = (float)(target - pos)/map.get(pos);
                if(carAheadTime < currCar) stack.push(pos);
            }
        }

        return stack.size();
    }
}
```
