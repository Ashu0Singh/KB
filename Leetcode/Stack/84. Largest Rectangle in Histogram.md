### Optimal Solution

So here we're gonna maintain a stack of index and heights and what we will do is check to what index an element can be extended. Rule of extending is that, if you get an element lesser than the current one you can't extend any further and you will have to pop it. But the idea is that if you're gonna pop it you know that the current element is lesser than the one on top of the stack. That means you can extent the current element backwards as well, so you're gonna pick the index of the element you're popping because you can extend the current element backward. Now every time you do this, just calculate the max area you can accommodate, i.e the element being popped * current index - their index. And compare it with previous maxima.

```java
class Solution {
    public class DS{
        int height;
        int index;
        DS(int h, int i){
            height = h;
            index = i;
        }
    }

    public int largestRectangleArea(int[] heights) {
        Stack<DS> stack = new Stack<>();
        int maxArea = 0;

        for(int i = 0; i < heights.length; i++){
            if(stack.isEmpty()) {
                stack.push(new DS(heights[i], i));
            } else {
                if(stack.peek().height <= heights[i]){
                    stack.push(new DS(heights[i], i));
                } else {
                    DS latest = null;
                    while(!stack.isEmpty() && stack.peek().height > heights[i]){
                        latest = stack.peek();
                        int area = (i - latest.index) * latest.height;
                        maxArea = Math.max(maxArea, area);
                        stack.pop();
                    }
                    stack.push(new DS(heights[i], latest.index));
                }
            }
        }

        while(!stack.isEmpty()){
            DS top = stack.pop();
            int area = (heights.length - top.index) * top.height;
            maxArea = Math.max(maxArea, area);
        }

        return maxArea;
    }
}
```
