### Optimal Approach

Push the opening brackets to the stack and then check if the upcoming character is a corresponding closing bracket or not, if not then return false. Also check if the stack is empty and incase it is return false. Also if stack has any characters after the string is exhausted then return false. 

```java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack = new Stack<>();

        for( char ch : s.toCharArray() ){
            if(ch == '(' || ch == '{' || ch == '[')
                stack.push(ch);
            else if (stack.isEmpty()) return false;
            else if ((ch == ')' && stack.peek() == '(') || 
                     (ch == '}' && stack.peek() == '{') ||
                     (ch == ']' && stack.peek() == '['))
                stack.pop();
            else return false;
        }
        if(stack.size() > 0) return false;
        return true;
    }
}
```
