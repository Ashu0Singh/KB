[Question](https://leetcode.com/problems/daily-temperatures/)

### Optimal Solution

Simply push the temperature's index to the stack and since you're only popping when a lower number shows up it will become a monotonic stack. Where you'll have to only check the top and if it's lesser than the current element pop and check the next until there's not element greater than the current element while simultaneously updating the answers array.

```java
class Solution {
    public int[] dailyTemperatures(int[] temperatures) {
        Stack<Integer> stack = new Stack<>();
        int[] answers = new int[temperatures.length];

        for(int i = 0; i < temperatures.length; i++){
	        answers[i] = 0;
            if(stack.isEmpty() || temperatures[stack.peek()] >= temperatures[i]) stack.push(i);
            else {
                int index = stack.pop();
                answers[index] = i - index;
                i--;
            }
        }

        return answers;
    }
}
```
