### Optimal Solution

Simply maintain an array along with the stack to make sure that you can easily refer to that whenever you're asked for the min value and then use the stack to peek for the top and pop the top.

```java
class MinStack {
    Stack<Integer> stack;
    ArrayList<Integer> minStack;
    public MinStack() {
        stack = new Stack<>();
        minStack = new ArrayList<>();
    }
    
    public void push(int val) {
        if(stack.isEmpty()){
            minStack.add(val);
        } else {
            minStack.add(Math.min(val, minStack.get(stack.size() - 1)));
        }
        stack.push(val);

    }
    
    public void pop() {
        stack.pop();
        minStack.remove(stack.size());
    }
    
    public int top() {
        return stack.peek();
    }
    
    public int getMin() {
        return minStack.get(stack.size() - 1);
    }
}

```
