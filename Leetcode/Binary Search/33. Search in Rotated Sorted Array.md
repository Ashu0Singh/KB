### Optimal Solution

Here's what we are assuming, so if the array is rotated either the first half or the second half would be sorted. And with whichever part is sorted we can use that to decide which direction we have to go.

```java
class Solution {
    public int search(int[] nums, int target) {
        int start = 0, end = nums.length - 1;
        while( start <= end){
            int mid = start + ((end - start) / 2);
            if(nums[mid] == target) return mid;
            if (nums[start] <= nums[mid]){
                if(nums[start] <= target && target < nums[mid])
                    end = mid - 1;
                else start = mid + 1;
            }
            else if (nums[mid] <= nums[end]){
                if(nums[mid] < target && target <= nums[end])
                    start = mid + 1;
                else end = mid - 1;
            }
        }
        return -1;
    }
}
```
