### Optimal Solution

It's pretty simple here, we're gonna create a Pair and then use it to set timestamp and value together in an hashmap. This will help us use less storage and be efficient with the operations. 
Since the question already specifies that it will be in strictly increasing order. We can apply binary search on the timestamp array and get done with it. 

```java
class TimeMap {

    static class Pair {
        int timestamp;
        String value;

        Pair(int t, String v) {
            timestamp = t;
            value = v;
        }
    }

    HashMap<String, ArrayList<Pair>> map;

    public TimeMap() {
        map = new HashMap<>();
    }

    public void set(String key, String value, int timestamp) {
        map.computeIfAbsent(key, k -> new ArrayList<>())
           .add(new Pair(timestamp, value));
    }

    public String get(String key, int timestamp) {
        ArrayList<Pair> list = map.get(key);
        if (list == null) return "";

        int start = 0;
        int end = list.size() - 1;
        String result = "";

        while (start <= end) {
            int mid = start + (end - start) / 2;

            if (list.get(mid).timestamp <= timestamp) {
                result = list.get(mid).value;
                start = mid + 1;
            } else {
                end = mid - 1;
            }
        }

        return result;
    }
}
```
