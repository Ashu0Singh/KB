### Optimal Solution

So we know that the least speed koko can eat at is 1 and max is the maximum of the array. We're gonna run a binary search on 1 -> max so that we can get a point where the number of hours are equal to the hours guards are gone.

```java
class Solution {
    public int minEatingSpeed(int[] piles, int h) {
        int max = 0;
        for(int pile : piles) max = Math.max(pile,max);

        int start = 1, end = max;
        int sol = max;
        while(start <= end){
            int mid = start + (end - start) / 2;
            long hours = 0;
            for(int pile : piles) {
                hours += (pile + mid - 1L) / (long)mid;
            }
            if(hours <= h) {
                sol = mid;
                end = mid - 1;
            } else {
                start = mid + 1;
            }
        }
        System.gc();
        return sol;
    }
}
```
