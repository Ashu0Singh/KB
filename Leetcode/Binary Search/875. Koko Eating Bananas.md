### Optimal Solution

So we know that the least speed Koko can eat at is 1 and max is the maximum of the array. We're gonna run a binary search on 1 -> max so that we can get a point where the number of hours are equal to the hours guards are gone.

This equation calculates the number of hours it takes Koko to eat a pile at a given speed.
The `+ mid - 1` trick does two things: 
1. it forces the result to be a long type, and
2. it achieves ceiling division - if the pile is perfectly divisible by mid, it returns that value; otherwise, it returns the ceiling of the division.

```java
hours += (pile + mid - 1L) / (long)mid;

hours += ceil(3/4)  = 1
hours += ceil(6/4)  = 2
hours += ceil(7/4)  = 2
hours += ceil(11/4) = 3
```

```java
class Solution {
    public int minEatingSpeed(int[] piles, int h) {
        int max = 0;
        for(int pile : piles) max = Math.max(pile,max);

        int start = 1, end = max;
        int sol = max;
        while(start <= end){
            int mid = start + (end - start) / 2;
            long hours = 0;
            for(int pile : piles) {
                hours += (pile + mid - 1L) / (long)mid;
            }
            if(hours <= h) {
                sol = mid;
                end = mid - 1;
            } else {
                start = mid + 1;
            }
        }
        System.gc();
        return sol;
    }
}
```
