### Optimal Solution

So the best way to search in a rotated array is simply binary search and collect the minimum element in a number.

```java
class Solution {
    public int findMin(int[] nums) {
        int start = 0, end = nums.length - 1;
        int res = nums[0];
        
        while(start <= end){
            int mid = start + ((end - start)/2);
            res = Math.min(res, nums[mid]);
            if(nums[mid] > nums[end]) start = mid + 1;
            else end = mid - 1;
        }
        return res;
    }
}
```
