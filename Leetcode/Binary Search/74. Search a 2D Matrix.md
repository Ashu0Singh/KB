[Question](https://leetcode.com/problems/search-a-2d-matrix/)
### Optimal Solution
The optimal way to solve this problem is to treat the 2 matrix like a linear array and since the matrix is sorted it makes it very easy to perform binary search on it

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length, n = matrix[0].length;
        int start = 0, end = (m * n) - 1;

        while(start <= end){
            int mid = start + ((end - start)/2);
            int element = matrix[mid/n][mid%n];
            if(element < target) start = mid + 1;
            else if(element > target) end = mid - 1;
            else return true;
        }

        return false;
    }
}
```
