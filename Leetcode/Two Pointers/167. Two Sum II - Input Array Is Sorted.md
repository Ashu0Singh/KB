[Question](https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/description/)
### Problem Explanation

Given a **sorted array** and a **target sum**, find two numbers that add up to the target. Return their **1-indexed positions** (not values).

**Key constraint:** The array is already sorted, which we can leverage for an efficient solution.

### Approach: Two Pointers

Since the array is sorted, use pointers at both ends:

1. `start` pointer at the beginning (index 0)
2. `end` pointer at the end (last index)
3. Calculate sum of elements at both pointers
4. If sum equals target → return indices (add 1 for 1-indexed)
5. If sum is too small → move `start` right (need larger numbers)
6. If sum is too large → move `end` left (need smaller numbers)

### Example Walkthrough

**Example 1:** `nums = [2, 7, 11, 15]`, `target = 9`

```
start=0 (2)  end=3 (15)
sum = 2 + 15 = 17 > 9  →  move end left

start=0 (2)  end=2 (11)
sum = 2 + 11 = 13 > 9  →  move end left

start=0 (2)  end=1 (7)
sum = 2 + 7 = 9 == 9  ✓
Return [1, 2] (1-indexed positions)
```

**Example 2:** `nums = [2, 3, 4]`, `target = 6`

```
start=0 (2)  end=2 (4)
sum = 2 + 4 = 6 == 6  ✓
Return [1, 3]
```

**Example 3:** `nums = [-1, 0]`, `target = -1`

```
start=0 (-1)  end=1 (0)
sum = -1 + 0 = -1 == -1  ✓
Return [1, 2]
```

### Optimal Solution

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int start = 0, end = nums.length - 1;
        while(start < end){
            int sum = nums[start] + nums[end];

            if(sum == target) return new int[]{start + 1, end + 1};
            else if(sum < target) start++;
            else end--;
        }
        return new int[2];
    }
}
```
