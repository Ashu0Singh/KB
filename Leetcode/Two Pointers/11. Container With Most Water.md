[Question](https://leetcode.com/problems/container-with-most-water/description/)

### Problem Explanation

Given an array of heights, find **two lines** that form a container with the **maximum area** of water that can be trapped.

**Area formula:** `width × height` where:

- `width` = distance between two indices
- `height` = minimum of the two bar heights (water overflows at the shorter bar)

### Intuition: Why Two Pointers Works

Start with the **widest possible container** (leftmost and rightmost bars). This gives maximum width, but possibly suboptimal height.

**Key insight:** We must move inward (reduce width). The question is: which pointer to move?

- If we move the **taller bar** inward:
  - Width decreases
  - Height is still limited by the shorter bar
  - Area can only decrease or stay the same ✗

- If we move the **shorter bar** inward:
  - Width decreases
  - But we might find a taller bar
  - Area might increase! ✓

**Strategy:** Always move the pointer pointing to the **shorter bar**. This gives us a chance to find a taller bar that might compensate for the lost width.

### Example Walkthrough

**Input:** `height = [1, 8, 6, 2, 5, 4, 8, 3, 7]`

```
Indices:  0  1  2  3  4  5  6  7  8
Heights:  1  8  6  2  5  4  8  3  7

start=0, end=8:
  min(1, 7) = 1, width = 8
  Area = 1 × 8 = 8
  height[0]=1 < height[8]=7, move start to 1

start=1, end=8:
  min(8, 7) = 7, width = 7
  Area = 7 × 7 = 49 ✓ (better)
  height[1]=8 > height[8]=7, move end to 7

start=1, end=7:
  min(8, 3) = 3, width = 6
  Area = 3 × 6 = 18
  height[1]=8 > height[7]=3, move end to 6

start=1, end=6:
  min(8, 8) = 8, width = 5
  Area = 8 × 5 = 40
  height[1]=8 == height[6]=8, move either (let's move start)
  ...

Maximum Area Found: 49
```

### Optimal Solution

```java
class Solution {
    public static int maxArea(int[] height) {
        int start = 0, end = height.length - 1, max = 0;
        while(start < end){
            int min = Math.min(height[start],height[end]);
            max = Math.max(max, min * (end - start));

            if(height[start] > height[end]) end--;
            else start++;
        }
        return max;
    }
}
```
