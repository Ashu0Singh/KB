[Question](https://leetcode.com/problems/valid-palindrome/)

### Problem Explanation

Check if a string is a **valid palindrome** considering only **alphanumeric characters** (letters and digits), and **ignoring case**.

**Key Rules:**

- Only compare letters (a-z, A-Z) and digits (0-9)
- Ignore all spaces and special characters
- Case-insensitive comparison

### Approach: Two Pointers

Use two pointers starting from both ends and move inward:

1. Skip non-alphanumeric characters on the left
2. Skip non-alphanumeric characters on the right
3. Convert both to lowercase and compare
4. If mismatch found, return `false`
5. Move both pointers closer and repeat

### Example Walkthrough

**Example 1:** `s = "A man, a plan, a canal: Panama"`

```
Clean up (alphanumeric only, lowercase): "amanaplanacanalpanama"

Compare with two pointers:
Left (a) == Right (a) ✓
Left (m) == Right (m) ✓
Left (a) == Right (a) ✓
...
Result: true (it's a palindrome)
```

**Example 2:** `s = "race a car"`

```
Clean up: "raceacar"

Compare:
Left (r) == Right (r) ✓
Left (a) == Right (a) ✓
Left (c) != Right (c) -> Actually c == c ✓
Left (e) != Right (a) ✗
Result: false (not a palindrome)
```

**Example 3:** `s = " "`

```
No alphanumeric characters found
Result: true (empty string is a palindrome)
```

### Optimal Solution

```java
class Solution {
    public boolean isPalindrome(String s) {
        char[] ch = s.toCharArray();
        int start = 0, end = ch.length - 1;

        while(start < end) {
            if(ch[start] >= 'A' && ch[start] <= 'Z') ch[start] = (char)(ch[start] + 32);
            if(ch[end] >= 'A' && ch[end] <= 'Z') ch[end] = (char)(ch[end] + 32);

            if(isAlphaNumeric(ch[start]) && isAlphaNumeric(ch[end]) && ch[start] != ch[end])
               return false;
            else if(!isAlphaNumeric(ch[start])) start++;
            else if(!isAlphaNumeric(ch[end])) end--;
            else {
                start++;
                end--;
            }
        }

        return true;
    }
    private boolean isAlphaNumeric(char ch){
        return ((ch >= 'a' && ch <= 'z') || (ch >= '0' && ch <= '9'));
    }
}
```
