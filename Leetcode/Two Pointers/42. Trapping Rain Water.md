[Question](https://leetcode.com/problems/trapping-rain-water/description/)

### Problem Explanation

Given an elevation map represented as an array of heights, calculate how much **water can be trapped** after it rains.

**Key insight:** Water at position `i` is trapped between the maximum heights on its left and right.

### Intuition

Water trapped at position `i` = `min(maxHeightLeft, maxHeightRight) - height[i]`

**Why?**

- Water level at position `i` is limited by the shorter of the two surrounding maximum heights
- If both sides are 5 and the bar at `i` is 2, water fills to level 5, trapping 3 units
- If there's no bar taller on one side, water flows out (0 trapped)

**Algorithm:**

1. Precompute `maxLeft[i]` = maximum height from index 0 to i-1
2. Precompute `maxRight[i]` = maximum height from index i+1 to end
3. For each position, calculate trapped water = `min(maxLeft[i], maxRight[i]) - height[i]`
4. Sum all trapped water

### Code Explanation

**Building maxLeft array:**

- `maxLeft[0] = 0` (no element to the left)
- `maxLeft[i] = max(maxLeft[i-1], height[i-1])` (max of previous max and previous element)

**Building maxRight array:**

- `maxRight[n-1] = 0` (no element to the right)
- `maxRight[i] = max(maxRight[i+1], height[i+1])` (max of next max and next element)

**Calculating trapped water:**

- For each position: `water = min(maxLeft[i], maxRight[i]) - height[i]`
- Only count if result is positive (not all positions trap water)

### Optimal Solution

```java
class Solution {
    public int trap(int[] h) {
        int l = h.length, count = 0;
        int[] maxLeft = new int[l], maxRight = new int[l];

        for(int i = 0; i < l; i++){
            if(i == 0) {
                maxLeft[i] = 0;
                maxRight[l - i - 1] = 0;
            } else {
                maxLeft[i] = Math.max(maxLeft[i - 1], h[i - 1]);
                maxRight[l - i - 1] = Math.max(maxRight[l - i], h[l - i]);
            }
        }

        for(int i = 0; i < l; i++){
            int min = Math.min(maxLeft[i], maxRight[i]);
            int water = min - h[i];

            if(water > 0) count += water;
        }

        return count;
    }
}
```
