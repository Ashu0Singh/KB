### Optimal Solution

This will be same as [[200. Number of Islands]] but in this case we'll count the area of each island and make sure the we compare it against other islands to get the maximum. 

```java
class Solution {
    public int maxAreaOfIsland(int[][] grid) {

        int max = 0;
        for(int row = 0; row < grid.length; row++){
            for(int col = 0; col < grid[0].length; col++){
                if(grid[row][col] == 1){
                    max = Math.max(dfs(grid, row, col), max);
                }
            }
        }
        return max;
        
    }
    int dfs(int[][] grid, int row, int col){
        if(row < 0 || col < 0 || row >= grid.length || col >= grid[0].length || grid[row][col] != 1) return 0;
        
        grid[row][col] = -1;
        int area = 1;
        area += dfs(grid, row + 1, col);
        area += dfs(grid, row, col + 1);
        area += dfs(grid, row, col - 1);
        area += dfs(grid, row - 1, col);

        return area;
    }
}
```
