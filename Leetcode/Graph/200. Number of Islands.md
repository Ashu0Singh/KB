### Optimal Solution

We will perform a DFS here, We will start with going through the Grid sequentially, but as soon as we encounter a land mass we will run a DFS on it and mark the visited cell as A so that we don't revisit it. With this we'll be able to get all the islands and won't revisit any cells!

```java
class Solution {
    public int numIslands(char[][] grid) {
        int count = 0;
        for(int row = 0; row < grid.length; row++){
            for(int col = 0; col < grid[0].length; col++){
                if(grid[row][col] == '1'){
                    dfs(grid, row, col);
                    count++;
                }
            }
        }
        return count;
    }

    void dfs(char[][] grid, int row, int col){
        if(row < 0 || col < 0 || row >= grid.length || col >= grid[0].length) return;
        
        if(grid[row][col] != '1') return;

        grid[row][col] = 'A';
        dfs(grid, row + 1, col);
        dfs(grid, row, col + 1);
        dfs(grid, row, col - 1);
        dfs(grid, row - 1, col);
    }
}
```
