### Optimal Solution

As mentioned in the question we need to complete all the courses. So what we're gonna do is treat the prerequisites arrays as the list of edges and create an [adjacency list](https://www.geeksforgeeks.org/dsa/adjacency-list-meaning-definition-in-dsa/) out of it. Once we have that, we're gonna perform a DFS traversal on every node where there are edges present. Because having edges mean that the course has prerequisites. Once the execution is done, we'll remove all the edges going outwards from the node and assume that the it has no more prerequisites, since all the prerequisite courses are successfully completed. This will help use track visited nodes overall.

```java
class Solution {
    public static boolean canFinish(int numCourses, int[][] prerequisites) {
        ArrayList<Integer>[] map = new ArrayList[numCourses];;
        int edges = prerequisites.length;

        // Creating the Adjacency List
        for(int[] edge : prerequisites){
            if(map[edge[0]] == null) map[edge[0]] = new ArrayList<>();
            map[edge[0]].add(edge[1]);
        }

        for(int i = 0; i < numCourses; i++){
            if(map[i] != null && !dfs(map, i, new boolean[numCourses])) 
                return false;
        }

        return true;
    }

    private static boolean dfs(List<Integer>[] map, int i, boolean[] visited){
        if(map[i] == null) return true;
        if(visited[i] == true) return false;

        /**
        The visited array is to detect a cycle. 
        So let's say if the path contains a certain element we should make sure that
        we are tracking it.
         */
        visited[i] = true;
        List<Integer> edgeList = map[i];
        
        boolean answer = true;
        for(int edge : edgeList){
            answer = answer && dfs(map, edge, visited);
        }

        map[i] = null;

        /**
        Since we don't want the branches to be included in the path we need
        to trim the off by setting visited[i] to false once the execution
        for that branch is completed.
         */
        visited[i] = false;
        return answer;
    }
}
```
