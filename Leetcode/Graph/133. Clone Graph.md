### Optimal Solution

We'll need a map to keep track of all the new nodes with the old ones. Once we have a new node created, we'll make sure to enrich it with the values we need. We'll also need to check if the node is already present in out Hashmap in order to avoid cycles!

```java
class Solution {
    HashMap<Node, Node> map = new HashMap<>();
    public Node cloneGraph(Node node) {
        if(node == null) return null;

        ArrayList<Node> copyList = new ArrayList<>();
        Node copyNode = new Node(node.val);
        map.put(node, copyNode);

        if(node.neighbors.size() > 0){
            for(Node currNode : node.neighbors){
                if(!map.containsKey(currNode)){
                    cloneGraph(currNode);
                } 
                copyList.add(map.get(currNode));
            }
        }
        if(copyList.size() > 0){
            copyNode.neighbors = copyList;
        }

        return copyNode;
    }
}
```
