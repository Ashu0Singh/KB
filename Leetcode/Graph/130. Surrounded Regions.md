[Question](https://leetcode.com/problems/surrounded-regions/description/)

### Brute force solution

```java
class Solution {
    boolean hasEdge;
    boolean[][] visited;
    int rows, cols;
    int[][] directions = {{1, 0}, {0, 1}, {-1, 0}, {0, -1}};
    public void solve(char[][] board) {
        rows = board.length;
        cols = board[0].length;
        visited = new boolean[rows][cols];

        for(int i = 0; i < rows; i++){
            for(int j = 0; j < cols; j++){
                List<int[]> list = new ArrayList<>();
                hasEdge = false;
                if(board[i][j] == 'O' && !visited[i][j]){
                    dfs(i,j,board, list);
                    if(!hasEdge){
                        for(int[] cord : list){
                            board[cord[0]][cord[1]] = 'X';
                        }
                    }
                } 
            }
        }
    }

    void dfs(int i, int j, char[][] board, List<int[]> list){
        visited[i][j] = true;
        list.add(new int[]{i, j});

        for(int[] d : directions){
            int x = i + d[0], y = j + d[1];

            if(x < 0 || y < 0 || x >= rows || y >= cols){
                hasEdge = true;
                continue;
            }
            if(board[x][y] == 'X') continue;
            if(visited[x][y]) continue;

            dfs(x, y, board, list);
        }
    }
}
```

### Optimal Solution

[[417. Pacific Atlantic Water Flow]] just like we did in this question we're gonna take all the `O`'s that are on the edges of the board and then run a DFS on each and mark it as some other character. Once we do that we'll only have the `O` that are not connected to any edges. No we can just turn the remaining `O`s into `X` and the random character we introduced to `O` and we should be good.

```java
class Solution {
    int m, n;
    public void solve(char[][] board) {
        m = board.length;
        n = board[0].length;
        
        for(int i = 0; i < board.length; i++){
            dfs(board, i, 0);
            dfs(board, i, board[0].length-1);
        }

        for(int i = 0; i < board[0].length; i++){
            dfs(board, 0, i);
            dfs(board, board.length-1, i);
        }
        convert(board,'O','X');
        convert(board,'#','O');
    }

    public void dfs(char[][] board, int nx, int ny){
        if(nx<0 || nx>=board.length || ny<0 || ny>=board[0].length || board[nx][ny]!='O')return;
        board[nx][ny]='#';
        dfs(board, nx-1, ny);
        dfs(board, nx+1, ny);
        dfs(board, nx, ny-1);
        dfs(board, nx, ny+1);
    }

    public void convert(char[][] board, char oldval, char newval){
        for(int i=0;i<board.length;i++){
            for(int j=0;j<board[0].length;j++){
                if(board[i][j]==oldval){
                    board[i][j]=newval;
                }
            }
        }
    }
}
```
