### Optimal Solution 

So we can solve this problem a couple of ways but the one mentioned down is efficient and in O(row * column) time complexity. So as we know that the outer most rows and columns are the ones that would be able to drain water to pacific or atlantic. 
With that knowledge we're gonna start the DFS inwards, what this means is that if the path is in increasing order or even equal, then we can consider all the elements in that path to be able to drain water to the pacific ocean or the atlantic ocean. 
Here we have 2 arrays on to track drainage for Pacific and the other one to track drainage for Atlantic. 

Now we'll take the top and left edge and run the DFS on each element to find all the cells that can drain to the pacific. And use the pacific array to mark the elements that can drain to the pacific. If we can reach a cell that means it can drain to pacific. No complex conditions required. Just make sure that the path is equal or increasing. Once we do this we'll have all the cells that can drain to the pacific. We'll perform the same for bottom and the right edge to get a similar result for the atlantic. After this we'll check if an element can drain to both and if yes push it to the array.

```java
class Solution {
    int rows, columns;
    int[][] directions = {{1,0},{-1,0},{0,1},{0,-1}};
    
    public List<List<Integer>> pacificAtlantic(int[][] heights) {
        this.rows = heights.length;
        this.columns = heights[0].length;
        boolean[][] pacific = new boolean[rows][columns];
        boolean[][] atlantic = new boolean[rows][columns];

        for(int i = 0; i < rows; i++) {
            dfs(i , 0 , heights , pacific );
            dfs(i, columns-1 , heights , atlantic );
        }
        for(int j = 0; j < columns; j++){
            dfs(0 , j , heights , pacific );
            dfs(rows-1, j , heights , atlantic);
        }

        List<List<Integer>> result = new ArrayList<>();

        for(int i = 0; i < rows; i++){
            for(int j = 0; j < columns; j++){
                if(pacific[i][j] && atlantic[i][j]){
                    result.add(Arrays.asList(i,j));
                }
            }
        }
        return result;
    }

    void dfs(int i, int j, int[][] heights, boolean[][] visited){
        if(visited[i][j] == true) return;
        visited[i][j] = true;

        for(int[] direction : directions){
            int x = i + direction[0], y = j + direction[1];

            if(x < 0 || y < 0 || x >= rows || y >= columns) continue;
            if(heights[i][j] > heights[x][y]) continue;

            dfs(x, y, heights, visited);
        }
    }
}
```

Another one with less arrays involved.

```java
class Solution {
    List<List<Integer>> result = new ArrayList<>();
    int rows, columns;
    int[][] directions = {{1,0},{-1,0},{0,1},{0,-1}};
    
    public List<List<Integer>> pacificAtlantic(int[][] heights) {
        this.rows = heights.length;
        this.columns = heights[0].length;
        char[][] visited = new char[rows][columns];

        for(int i = 0; i < rows; i++) {
            dfs(i , 0 , heights , visited, 'P');
        }
        for(int j = 0; j < columns; j++){
            dfs(0 , j , heights , visited, 'P');
        }
        for(int i = 0; i < rows; i++) {
            dfs(i, columns-1 , heights , visited , 'A');
        }
        for(int j = 0; j < columns; j++){
            dfs(rows-1, j , heights , visited , 'A');
        }

        return result;
    }

    void dfs(int i, int j, int[][] heights, char[][] visited, char ocean){
        if(visited[i][j] == 'P' && ocean == 'A'){
            result.add(Arrays.asList(i,j));
        }

        visited[i][j] = ocean;

        for(int[] direction : directions){
            int x = i + direction[0], y = j + direction[1];

            if(x < 0 || y < 0 || x >= rows || y >= columns) continue;
            if(heights[i][j] > heights[x][y]) continue;
            if(visited[x][y] == ocean) continue;

            dfs(x, y, heights, visited, ocean);
        }
    }
}
```
