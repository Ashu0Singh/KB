[Question](https://leetcode.com/problems/decode-ways/description/)

### Optimal Solution

This solution uses **recursive memoization** to count all possible ways to decode a string of digits. The key insight is that at each position, we can decode the current digit as a single character or combine it with the next digit as a two-digit character.

**Key Idea:**
Numbers 1-26 map to letters A-Z. For each position, we have two choices:

1. **Decode as single digit**: Move to position i + 1 (if digit is 1-9)
2. **Decode as two digits**: Move to position i + 2 (if digits form 10-26)

**Algorithm:**

1. **Base case**: If we reach the end (index >= length), we've found one valid decoding
2. **Invalid case**: If current digit is '0', return 0 (can't decode single 0)
3. **Memoization check**: Return cached result if already computed
4. **Decode as single digit**: `map[index] = helper(ch, index + 1, map)` — count ways from next position
5. **Try two-digit decoding**: If current and next digit form 10-26, add ways from position i+2
6. **Cache and return**: Store result in memoization array

**Example**: `s = "226"`

```java
- helper(0, '2'):
  - Single: '2' → helper(1, '2')
  - Double: '22' → helper(2, '6') = 1 (base case)
- helper(1, '2'):
  - Single: '2' → helper(2, '6') = 1
  - Double: '26' → helper(3) = 1 (base case)
  - Total: 2
- Result: 1 + 2 = **3 ways**: "2 2 6", "22 6", "2 26"
```


**Time Complexity**: O(n) — each position computed once due to memoization  
**Space Complexity**: O(n) — memoization array + recursion call stack


```java
class Solution {
    public int numDecodings(String s) {
        return helper(s.toCharArray(), 0, new Integer[s.length()]);
    }

    int helper(char[] ch, int index, Integer[] map){
        if(index >= ch.length) return 1;
        if((ch[index] - '0') == 0) return 0;
        if(map[index] != null) return map[index];

        map[index] = helper(ch, index + 1, map);

        if(index < ch.length - 1){
            int doubleDigit = (ch[index] - '0') * 10 + (ch[index + 1] - '0');

            if(doubleDigit > 9 && doubleDigit < 27)
                map[index] += helper(ch, index + 2, map);
        }

        return map[index];
    }
}
```
