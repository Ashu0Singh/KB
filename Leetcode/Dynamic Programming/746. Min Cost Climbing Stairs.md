### Optimal Solution
So the explanation here is simple at every step we will evaluate that which will incur the least cost for us in previous 2 cells. Let's say our current position is n so what we need to figure out is that what will get use the least cost n - 1 or n - 2. 

We will start from 2 as min of 0 - 1 and 0 - 2 will give use 0 and same goes for 1 as we're considering anything < 0 to be 0 cost. 

| index |  0  |  1  |  2  |  3  |  4  |  5  |  6  |  7  |  8  |
| ----- | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
| value |  1  | 100 |  1  |  1  |  1  | 100 |  1  |  1  | 100 |
| min   |  1  | 100 |  2  |  3  |  3  | 103 |  4  |  5  | 104 |
```java
class Solution {
    public int minCostClimbingStairs(int[] cost) {
        for(int i = 2; i < cost.length; i++){
            cost[i] += Math.min(cost[i-1], cost[i-2]);
        }
        return Math.min(cost[cost.length-1], cost[cost.length-2]);
    }
}
```
