### Optimal Solution

So this is how its gonna look, kind off.

| n     | 1   | 2   | 3   | 4   | 5   | 6   |
| ----- | --- | --- | --- | --- | --- | --- |
| steps | 1   | 2   | 3   | 5   | 8   | 13  |
Now you can solve it in two was, using a mem array, or just by generating the Fibonacci sequence where `an = an-1 + an-1` and `a0` = 1 , `a1` = 1. 

```java
class Solution {
    int[] mem;
    Solution(){
        mem = new int[46];
        for(int i = 0; i < 46; i++)
            mem[i] = -1;
    }
    public int climbStairs(int n) {
        if( n < 0) return 0;
        if( n == 0 ) return 1;

        if(mem[n] != -1) return mem[n];

        mem[n] = climbStairs(n - 1) + climbStairs(n - 2);

        return mem[n];
    }
}
```
