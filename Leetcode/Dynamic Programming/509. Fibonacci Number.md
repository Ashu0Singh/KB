### Optimal Solution

This solution computes the Fibonacci number using **space-optimized dynamic programming**. Instead of storing all previous Fibonacci numbers, it only keeps track of the last two values.

**Key Idea:**
The Fibonacci sequence follows: F(n) = F(n-1) + F(n-2). At each step, we only need the previous two numbers to compute the current one.

**Algorithm:**

1. **Base cases**: 
   - If n = 0, return 0 (fib(0) = 0)
   - If n = 1, return 1 (fib(1) = 1)
2. **Initialize pointers**:
   - `prev1 = 0` (fib(0))
   - `prev2 = 1` (fib(1))
3. **Iterate from 2 to n**:
   - `curr = prev1 + prev2` → Calculate current Fibonacci number
   - `prev1 = prev2` → Slide: second-last becomes last
   - `prev2 = curr` → Slide: current becomes second-last
4. **Return**: `curr` (the nth Fibonacci number)

**Example**: `n = 5`
- i=2: curr = 0+1 = 1, prev1=1, prev2=1 (fib(2)=1)
- i=3: curr = 1+1 = 2, prev1=1, prev2=2 (fib(3)=2)
- i=4: curr = 1+2 = 3, prev1=2, prev2=3 (fib(4)=3)
- i=5: curr = 2+3 = 5, prev1=3, prev2=5 (fib(5)=5)
- Result: 5

**Time Complexity**: O(n) — single loop from 2 to n  
**Space Complexity**: O(1) — only three variables regardless of input size

```java
class Solution {
    public int fib(int n) {
        if(n == 0) return 0;
        if(n == 1) return 1;
        int prev1 = 0;
        int prev2 = 1;
        int curr = 0;
        for(int i = 2; i <= n; i++){
            curr = prev1 + prev2;
            prev1 = prev2;
            prev2 = curr;
        }
        return curr;
    }
}
```
