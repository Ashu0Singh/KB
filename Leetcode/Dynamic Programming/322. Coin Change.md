[Question](https://leetcode.com/problems/coin-change/description/)

### Problem Explanation

Given an array of coin denominations and a target amount, find the **minimum number of coins** needed to make that amount. Return `-1` if it's impossible.

**Key insight:** This is a classic DP problem where we build up solutions from smaller amounts to the target amount.

### Approach: Bottom-Up DP

We create a DP array where `dp[i]` represents the minimum coins needed to make amount `i`.

**Steps:**

1. Initialize `dp[0] = 0` (0 coins needed for amount 0)
2. For each amount from 1 to target:
   - For each coin denomination:
     - If coin ≤ amount, we can use this coin
     - Try using this coin: `dp[amount] = min(dp[amount], dp[amount - coin] + 1)`
3. Return `dp[target]` or `-1` if still unreachable

### Example Walkthrough

**Input:** `coins = [1, 2, 5]`, `amount = 5`

```
dp array (index = amount, value = min coins):
dp[0] = 0       (0 coins for amount 0)
dp[1] = 1       (1×1 coin)
dp[2] = 1       (1×2 coin)
dp[3] = 2       (1×2 + 1×1)
dp[4] = 2       (2×2)
dp[5] = 1       (1×5 coin) ← Answer
```

**Another example:** `coins = [2]`, `amount = 3`

- Can't make 3 with only 2-coin denominations → return `-1`

### Optimal Solution

```java
class Solution {
    public int coinChange(int[] coins, int amount) {
        // dp[i] = minimum coins needed to make amount i
        int[] dp = new int[amount + 1];

        // Initialize: all amounts unreachable except 0
        for (int i = 1; i <= amount; i++) {
            dp[i] = Integer.MAX_VALUE;
        }
        dp[0] = 0;

        // For each amount from 1 to target
        for (int i = 1; i <= amount; i++) {
            // Try each coin
            for (int coin : coins) {
                if (coin <= i && dp[i - coin] != Integer.MAX_VALUE) {
                    dp[i] = Math.min(dp[i], dp[i - coin] + 1);
                }
            }
        }

        return dp[amount] == Integer.MAX_VALUE ? -1 : dp[amount];
    }
}
```

**Time Complexity:** O(amount × coins.length)  
**Space Complexity:** O(amount)
