[Question](https://leetcode.com/problems/palindromic-substrings/)
### Optimal Solution

This solution uses the **Expand Around Center** approach to find all palindromic substrings. Instead of checking every possible substring, it treats each position (and pair of positions) as a potential palindrome center and expands outward.

**Key Idea:**
Every palindrome has a center:
- **Odd-length palindromes**: Center is a single character (e.g., "aba" centered at 'b')
- **Even-length palindromes**: Center is between two characters (e.g., "abba" centered between the two 'b's)

For each position, we check both cases and count palindromes found.

**Algorithm:**

1. **Convert string to char array**: Easier to access characters by index
2. **For each position i**:
   - **Check odd-length palindromes**: `expand(ch, i, i)` — expand from single character center
   - **Check even-length palindromes**: `expand(ch, i, i+1)` — expand from two-character center
   - Add both counts
3. **Helper function `expand`**:
   - Start with `left` and `right` pointers
   - While characters match and pointers are in bounds:
     - Increment count (palindrome found)
     - Move pointers outward (`left--`, `right++`)
   - Return total palindromes found with this center

**Example 1**: `s = "abc"`

- **i = 0** (center at 'a'):
  - Odd: expand(0, 0) → 'a' matches → count = 1
  - Even: expand(0, 1) → 'a' ≠ 'b' → count = 0

- **i = 1** (center at 'b'):
  - Odd: expand(1, 1) → 'b' matches → count = 1
  - Even: expand(1, 2) → 'b' ≠ 'c' → count = 0

- **i = 2** (center at 'c'):
  - Odd: expand(2, 2) → 'c' matches → count = 1
  - Even: expand(2, 3) → out of bounds → count = 0

**Result**: 3

---

**Example 2**: `s = "aab"`

- **i = 0**: Odd→"a"(1), Even→skip
- **i = 1**: Odd→"a"(1) + expand even→"aa"(1) = 2
- **i = 2**: Odd→"b"(1), Even→skip

**Result**: 4 palindromes: "a", "a", "aa", "b"

**Time Complexity**: O(n²) — n centers, each expand can go up to n characters  
**Space Complexity**: O(1) — only using pointers and count variable

```java
class Solution {
    public int countSubstrings(String s) {
        char[] ch = s.toCharArray();
        int count = 0;
        for(int i = 0; i < ch.length; i++){
            count += expand(ch, i, i+1) + expand(ch, i, i);
        }
        return count;
    }

    int expand(char[] s , int left, int right){
        int count = 0;
        while(left >= 0 && right < s.length && s[left] == s[right]) {
            left--; right++; count++;
        }
        return count;
    }
}
```
