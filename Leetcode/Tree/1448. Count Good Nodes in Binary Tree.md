### Optimal Solution

So to get the good nodes of a Binary tree we will pass down the max value we've encountered through the path and if the node value > max than it's a good node and pass the max down but if not. Then it's not a good node and we can move to the next node. 

```java
class Solution {
    int count;
    public int goodNodes(TreeNode root) {
        count = 0;
        goodNode(root, Integer.MIN_VALUE);
        return count;
    }

    private void goodNode(TreeNode node, int max){
        if(node == null) return;

        if( max <= node.val ) {
            max = node.val;
            count++;
        }

        goodNode(node.left, max);
        goodNode(node.right, max);
    }
}
```
