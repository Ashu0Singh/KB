### Optimal Solution

Here we know that the first value in an preorder traversal is our root node. So we're gonna start with building out tree by using a helper. We will have our preorder traversal, preorder index. When preorder index is 0, we're at the root node. So we will find the index of the root node in the inorder traversal using an hash map. The simple principle here is that anything to the left of the current node will in the LST and anything on the right would be in our RST. What we will do here is, run the function recursively.

```java
class Solution {
    int index = 0; // tracks position in preorder

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        HashMap<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < inorder.length; i++) {
            map.put(inorder[i], i); // map inorder value to its index
        }
        return helper(preorder, 0, inorder.length - 1, map); // Treating this as start of the BST since recurssion is gonna follow the same.
    }

    private TreeNode helper(int[] preorder, int start, int end, HashMap<Integer, Integer> map) {
        if (start > end) return null;

        int rootVal = preorder[index++];
        TreeNode node = new TreeNode(rootVal);

        int inorderIndex = map.get(rootVal);

        node.left = helper(preorder, start, inorderIndex - 1, map);
        node.right = helper(preorder, inorderIndex + 1, end, map);

        return node;
    }
}
```
