[Question](https://leetcode.com/problems/binary-tree-right-side-view/)

### Optimal Solution 

This one is similar to the last visited question of [[102. Binary Tree Level Order Traversal]] in a way that there we were storing the entire level in an array. But here, we will just store it as an element, and we'll start traversal from the right instead of left so that we will only get the right most values.

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> list = new ArrayList<>();
        helper(root, list, 0);
        return list;        
    }
    private void helper(TreeNode root, List<Integer> list, int index){
        if(root == null) return;

        if(list.size() == index) list.add(index, root.val);
        helper(root.right, list, index + 1);
        helper(root.left, list, index + 1);
    }
}
```
