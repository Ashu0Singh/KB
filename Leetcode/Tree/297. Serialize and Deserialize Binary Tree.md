[Question](https://leetcode.com/problems/serialize-and-deserialize-binary-tree/)

### Optimal Solution

It's very simple, see so we know that the termination node is = null. So we will do one thing, we will first place root, then place left sub tree and then right sub tree. And wherever we encounter null we will put an "n". What this will help us with is when to terminate the left sub tree and move to the right sub tree. 

```java
public class Codec {
    int index = 0;

    // Encodes a tree to a single string.
    public String serialize(TreeNode root) {
        if(root == null) return "n";

        String left = serialize(root.left);
        String right = serialize(root.right);

        return root.val + "," + left + "," + right ;
    }
    // Decodes your encoded data to tree.
    public TreeNode deserialize(String data) {
        return deserializer(data.split(","));
    }

    private TreeNode deserializer(String[] data){
        if(index >= data.length || data[index].equals("n") || data[index].isEmpty()) return null;

        TreeNode root = new TreeNode(Integer.parseInt(data[index]));
        index++;
        root.left = deserializer(data);
        index++;
        root.right = deserializer(data);
        return root;
    }
}
```
