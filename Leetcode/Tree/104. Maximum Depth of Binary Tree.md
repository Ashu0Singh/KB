[Question](https://leetcode.com/problems/maximum-depth-of-binary-tree/)

### Optimal Solution 

Go through every node and when you reach to the end return the position of the current node. Get the max of left and right node at every point and you'll get the max depth.

```java
class Solution {
    public int maxDepth(TreeNode root) {
        if(root == null) return 0;

        int left = maxDepth(root.left);
        int right = maxDepth(root.right);

        int currMd = Math.max(left,right);

        return currMd + 1;
    }
}
```
