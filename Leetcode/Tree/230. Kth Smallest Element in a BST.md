[Question](https://leetcode.com/problems/kth-smallest-element-in-a-bst/)
### Optimal Solution

We will keep a counter, since LST is the smallest value we will always look for the LST and decrease the count if we find the LST. Do the same until count == 0 set the result of the root values and disappear into the wild. 

```java
class Solution {
    int res;
    int count;
    public int kthSmallest(TreeNode root, int k) {
        this.count = k;
        this.res = -1;
        helper(root);
        return this.res;
    }

    private void helper(TreeNode root){
        if(root == null) return;

        helper(root.left);
        if(--count == 0) {
            res = root.val;
            return;
        }
        helper(root.right);
    }
}
```
