### Optimal Solution

Since it's a BST we can use the properties of a BST to get the lowestCommonAncestor. What we can do here is check if p and q are before. Or after the root node. If any of these cases are true that means we can say that the current node ain't the lowestCommonAncestor. If not that means the current node is the lowest common ancestor. 

```java
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root.val < p.val && root.val < q.val) 
            return lowestCommonAncestor(root.right, p, q);
        else if (root.val > p.val && root.val > q.val) 
            return lowestCommonAncestor(root.left, p, q);
        return root;
    }
}
```
