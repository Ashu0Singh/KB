[Question](https://leetcode.com/problems/validate-binary-search-tree/)

### Optimal Solution

To validate a BST we need to make sure that value of root should always be greater than any values in LST should always be lesser than any values in RST. So we will pass down, a range with min and max. Where a value would always have a range where it should be in. If a value is out of that range then return false. The root node would be maxima for the LST and the minima for the BST. 

```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        return check(root, Long.MIN_VALUE, Long.MAX_VALUE);
    }

    public static boolean check(TreeNode root, long min, long max){
        if(root == null) return true;
        if(root.val <= min || root.val >= max) return false;
        return check(root.left, min, root.val) && check(root.right, root.val, max);
    }
}
```
