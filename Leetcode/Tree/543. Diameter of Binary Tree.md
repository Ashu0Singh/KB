[Question](https://leetcode.com/problems/diameter-of-binary-tree/description/)
### Optimal Solution

Just get the max depth of right subtree and max depth left subtree and add it to get the maxDiameter. 

```java
class Solution {
    int maxDiameter;
    public int diameterOfBinaryTree(TreeNode root) {
        maxDepth(root);
        return maxDiameter;
    }
    private int maxDepth(TreeNode root) {
        if(root == null) return 0;

        int left = maxDepth(root.left);
        int right = maxDepth(root.right);
        maxDiameter = Math.max(maxDiameter, left + right);

        int currMd = Math.max(left,right);

        return currMd + 1;
    }
}
```
