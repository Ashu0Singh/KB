[Question](https://leetcode.com/problems/invert-binary-tree/)

### Optimal Solution

So you simply traverse through the BST and get the left and right value. Once you have that, just swap the values. Do it recursively and you should be good. 

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        if(root == null) return null;

        TreeNode left = invertTree(root.left);
        TreeNode right = invertTree(root.right);

        root.left = right;
        root.right = left;

        return root;
    }
}
```
