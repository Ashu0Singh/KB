### Optimal Solution

First we'll use fast and a slow pointer to detect the cycle once we have detected the cycle, we'll go start another slow pointer from the head and the point where both the pointers will intersect is our starting node of the linked list. 

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;
        while(fast != null && fast.next != null){
            slow = slow.next;
            fast = fast.next.next;
            if(fast == slow){
                slow = head;
                while(slow != fast){
                    slow = slow.next;
                    fast = fast.next;
                }
                return slow;
            }
        }
        return null;
    }
}
```
