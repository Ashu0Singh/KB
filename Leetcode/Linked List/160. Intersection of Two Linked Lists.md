[Question](https://leetcode.com/problems/intersection-of-two-linked-lists/)

### Optimal Solution

To find intersection just push all the items to a stack and then pop it all in reverse order and the point where nodes are not equal just return the previous element

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        Stack<ListNode> l1 = new Stack<ListNode>();
        Stack<ListNode> l2 = new Stack<ListNode>();
        ListNode pointer1 = headA;
        ListNode pointer2 = headB;

        while(pointer1 != null){
            l1.push(pointer1);
            pointer1 = pointer1.next;
        }
        while(pointer2 != null){
            l2.push(pointer2);
            pointer2 = pointer2.next;
        }
        ListNode prev = null;

        while(!(l1.isEmpty() || l2.isEmpty())){
            pointer1 = l1.pop();
            pointer2 = l2.pop();
            if(pointer1 == pointer2) prev = pointer1;
            else return prev;
        }

        return prev;
    }
}
```
