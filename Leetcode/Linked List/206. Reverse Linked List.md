### Optimal Solution

The best way to do this is using recursion, you just go till the end node and start returning that while simultaneously moving every node to the next of next, so that when you come back to the top you'll have the linked list reversed with the head node being the last node.

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        if(head == null || head.next == null) return head;
        ListNode newHead = reverseList(head.next);
        head.next.next = head;
        head.next = null;
        return newHead;
    }
}
```
