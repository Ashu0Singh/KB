### Optimal Approach

Since the question does not requires us to match the memory reference, we can actually overwrite the current node value with the next node value and delete the next node!

```java
class Solution {
    public void deleteNode(ListNode node) {
        node.val = node.next.val;
        node.next = node.next.next;
    }
}
```
