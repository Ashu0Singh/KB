### Optimal Solution

So what we're gonna do is first will use recursion to get to the last node and then start counting from there so that we can match the number of elements from last. Once n == count, we'll know that we have to remove the next element. 

To cover all the edge case we'll also have to create a new head node and then prefix it to the subarray. This will allow us to be able to delete event the old head node.

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode currHead = new ListNode();
        currHead.next = head;
        removeNthNode(currHead,n);
        return currHead.next;
    }

    public int removeNthNode(ListNode head, int n){
        if(head.next == null) return 1;

        int currentNode = removeNthNode(head.next,n);
        if(currentNode == n){
            ListNode temp = head.next.next;
            head.next.next = null;
            head.next = temp;
        }

        return currentNode + 1;
    }
}
```
