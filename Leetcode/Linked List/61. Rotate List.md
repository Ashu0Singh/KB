[Question](https://leetcode.com/problems/rotate-list/)

### Optimal Solution

Let's assume that the length of the linked list is `n` so we will see that after every n rotation the list is same.
What we can do here is do a modules operation to get the number of times we need to rotate. Once we have that we will pick the last k elements from the last and then append it to the starting of the array. That's all we're doing here.

```java
class Solution {
    public ListNode rotateRight(ListNode head, int k) {
        if(head == null) return head;

        int length = 1;
        ListNode lastPointer = head;
        while(lastPointer.next != null){
            lastPointer = lastPointer.next;
            length++;
        }

        k = k % length;

        int iter = length - k;
        ListNode start = head;

        while(--iter  > 0){
            start = start.next;
        }

        lastPointer.next = head;
        head = start.next;
        start.next = null;

        return head;
    }
}
```
