[Question](https://leetcode.com/problems/copy-list-with-random-pointer/)

### Optimal Solution

We will create a copy head pointer here and a pointer, what we're gonna do is. Simply create copied linked list and push all the real and copied nodes into a hashmap. Now reiterated through the real linked list and check the random pointer. If it has a random pointer, use the hashmap to get the value and push the value to the copied linked list. 

```java
/*
// Definition for a Node.
class Node {
    int val;
    Node next;
    Node random;

    public Node(int val) {
        this.val = val;
        this.next = null;
        this.random = null;
    }
}
*/

class Solution {
    public Node copyRandomList(Node head) {
        HashMap< Node , Node > map = new HashMap<>();
        Node copyhead = new Node(0);
        
        Node pointer = head;
        Node copyPointer = copyhead;

        while(pointer != null){
            Node temp = new Node(pointer.val);
            map.put(pointer, temp);
            copyPointer.next = temp;

            copyPointer = copyPointer.next;
            pointer = pointer.next;
        }

        pointer = head;

        while(pointer != null){
            Node temp = map.get(pointer);

            if(pointer.random != null){
                Node random = map.containsKey(pointer.random) ? map.get(pointer.random) : new Node(pointer.random.val);
                temp.random = random;
            }

            pointer = pointer.next;
        }

        return copyhead.next;
    }
}
```
