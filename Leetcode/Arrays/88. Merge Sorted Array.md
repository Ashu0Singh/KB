[Question](https://leetcode.com/problems/merge-sorted-array/)

### Optimal Solution 

Take the bigger array and start filling it from the last, so basically let’s say all the elements in array 2 is greater than all of them in array 1 then you would have filled all the elements in array 1 already.

```java
class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int num1Count = m - 1;
        int num2Count = n - 1;
        for(int i = m + n - 1; i >= 0; i--){
            if(num1Count < 0){
                nums1[i] = nums2[num2Count];
                num2Count--;
            }
            else if(num2Count < 0){
                nums1[i] = nums1[num1Count];
                num1Count--;
            }
            else if(nums1[num1Count] > nums2[num2Count]){
                nums1[i] = nums1[num1Count];
                num1Count--;
            }
            else {
                nums1[i] = nums2[num2Count];
                num2Count--;
            }
        }
    }
}
```
