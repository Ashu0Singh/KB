### Better Solution

We're creating a prefix array and population it with the prefix product, after that we're creating another array and populating it with suffix product. This helps us create, 2 arrays where we have product of prefix and suffix except self. After this we'll multiply the products to get the final answer except self. 

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int[] prefix = new int[nums.length];
        int product = 1;
        for(int i = 0; i < nums.length; i++){
            prefix[i] = product;
            product = product * nums[i];
        }
        int[] sufix = new int[nums.length];
        product = 1;
        for(int i = nums.length - 1; i >= 0; i--){
            sufix[i] = product;
            product = product * nums[i];
        }

        for(int i = 0; i < nums.length; i++){
            prefix[i] = prefix[i] * sufix[i];
        }

        return prefix;
    }
}
```
