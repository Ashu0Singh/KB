[Question](https://leetcode.com/problems/merge-intervals/)

### Optimal Solution

Just sort the array and then keep a track of last interval this will let you check whether the current interval is mergeable or not. And then merge whichever interval you could.

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        int n = intervals.length;
        Arrays.sort(intervals, (a,b) -> a[0] - b[0]);
        List<int[]> answer = new ArrayList<>();

        int[] prevInterval = intervals[0];

        for(int i = 0; i < n; i++){
            if(prevInterval[1] < intervals[i][0]){
                answer.add(prevInterval);
                prevInterval = intervals[i];
            } else {
                prevInterval[0] = intervals[i][0] < prevInterval[0] ? intervals[i][0] : prevInterval[0];
                prevInterval[1] = intervals[i][1] > prevInterval[1] ? intervals[i][1] : prevInterval[1];
            }
        }
        answer.add(prevInterval);

        return answer.toArray(new int[answer.size()][]);
    }
}
