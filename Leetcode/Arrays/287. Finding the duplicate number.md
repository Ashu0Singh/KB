[Question](https://leetcode.com/problems/find-the-duplicate-number/)

### Optimal Solution

Since the question explicitly asks use to not use extra space or modify the array, in this case it get very hard to detect the duplicate number.
So the idea is that since the question states that the number in the arrays is gonna be 1-n then we can see that if some index is being pointed to more than once that might be the duplicate index.

```java
class Solution {
    public int findDuplicate(int[] nums) {
        int fast = 0, slow = 0;
        do {
            fast = nums[nums[fast]];
            slow = nums[slow];
        } while(fast != slow);
        
        int slow2 = 0;
        while(slow != slow2){
            slow = nums[slow];
            slow2 = nums[slow2];
        }
        return slow;
    }
}
```
