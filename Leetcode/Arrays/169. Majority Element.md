### Brute Force Solution

To find the majority element we can simply count the occurrence of each element and check which ever is highest element we should just return that as the question states that every test case would have a majority element

```java
class Solution {
    public int majorityElement(int[] nums) {
        HashMap<Integer, Integer> countMap = new HashMap<>();
        int max = 0, element = nums[0];
        for(int num : nums){
            int count = countMap.getOrDefault(num, 0) + 1;
            countMap.put(num, count);
            if(max < count) {
                element = num; 
                max = count;
            }
        }
        return element;
    }
}
```
