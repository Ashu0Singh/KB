[Question](https://leetcode.com/problems/maximum-subarray/)
### Brute force Solution

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int length = nums.length;
        int max = nums[0];
        for(int i = 0; i < length; i++){
            int sum = 0;
            for(int j = i; j < length; j++){
                sum = sum + nums[j];
                max = sum > max ? sum : max;
            }
        }
        return max;
    }
}
```

### Optimal Solution

So the idea is that we do not want to carry forward any negative sum, as that will decrease the sum of the upcoming subarray where as if it’s positive it will increase the sum of the upcoming subarray.

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int max = nums[0], sum = 0;
        for(int i = 0; i < nums.length; i++){
            sum += nums[i];
            max = sum > max ? sum : max;
            if(sum < 0) sum = 0;
        }
        return max;
    }
}
```
