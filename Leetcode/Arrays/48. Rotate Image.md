[Question](https://leetcode.com/problems/rotate-image/)

### Brute force Solution
So the brute force way to do it is to create another array, and then take the first row, turn that into a column take the second row and turn it into another column, and at the end assign that to the current matrix and you’re good. 

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        int[][] answer = new int[n][n];

        for(int i = 0; i < n; i++){
            for(int j = 0; j < n; j++){
                answer[j][n - i - 1] = matrix[i][j];
            }
        }

        for(int i = 0; i < n; i++){
            for(int j = 0; j < n; j++){
                matrix[i][j] = answer[i][j];
            }
        }
    }
}
```

### Optimal Solution 
The simplest way to do it to transpose the matrix and then reverse all the rows one by one and you should have the rotated answer. 

```java
class Solution {
    public void rotate(int[][] matrix) {
        transpose(matrix);
        reverse(matrix);
    }
    private static void swap(int[][] matrix, int startRow, int startCol, int endRow, int endCol){
        int temp = matrix[startRow][startCol];
        matrix[startRow][startCol] = matrix[endRow][endCol];
        matrix[endRow][endCol] = temp;
        
    }
    private static void transpose(int[][] matrix){
        int n = matrix.length;
        for(int i = 0; i < n; i++){
            for(int j = i; j < n; j++){
                swap(matrix,i,j,j,i);
            }
        }
    }
    private static void reverse(int[][] matrix){
        int n = matrix.length - 1;
        for(int i = 0; i <= n; i++){
            for(int j = 0; j <= n/2; j++){
                swap(matrix,i,j,i,n - j);
            }
        }
    }
}
```
