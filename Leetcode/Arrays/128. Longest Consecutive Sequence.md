[Question](https://leetcode.com/problems/longest-consecutive-sequence/)

### Brute Force Approach

Just sort the Array using Arrays.sort and then use a loop to go through it and keep counting until the next element is  != current + 1. Reset the count and do it again and use comparison to find the highest count.

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        Arrays.sort(nums);
        if(nums.length == 0) return 0;

        int max = 1;
        int count = 1;
        for(int i = 1; i < nums.length; i++){
            if(nums[i - 1] == nums[i]) continue;
            else if(nums[i - 1] != (nums[i] - 1)) count = 1;
            else count++;

            if(count > max) max = count;
        }
        return max;
    }
}
```

### Optimal Approach

For this one, first of all add all the element to a set, and then go through the element of the set or the element of the array and for every element which doesn't has a previous element as in value - 1. Just start counting in the set and remove all the elements you have counted. And Once you're done counting compare it to the maximum. 

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for(int num : nums) set.add(num);

        int max = 0;
        for(int num : nums){
            if(!set.contains(num - 1)){
                int curr = num;
                int count = 0;
                while(set.contains(curr)){
                    set.remove(curr);
                    count++;
                    curr++;
                }
                if(count > max) max = count;
            }
        }
        return max;
    }
}
```
