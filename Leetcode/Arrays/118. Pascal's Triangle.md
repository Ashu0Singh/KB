### Optimal Solution

You can do it both ways, using 2D matrix or List of list. 2D matrix runs faster a it’s a primitive data structure. 

Here’s how we solve it, simply take the number of rows and then create a 2D matrix with that size now run a for loop from 0 -> number of rows. Every time you set the first and last element as 1. And then for all the middle elements use the previous list in the array and get the sum of element at current position in the array and the one before that and push it to the array.

```java
class Solution {
    public List<List<Integer>> generate(int numRows) {
        int[][] triangle = new int[numRows][];

        for(int i = 0; i < numRows; i++ ){
            triangle[i] = new int[i+1];
            triangle[i][0] = 1;
            triangle[i][i] = 1;
            for(int j = 1; j < i; j++){
                triangle[i][j] = triangle[i-1][j] + triangle[i-1][j-1];
            }
        }
        return (List)Arrays.asList(triangle);
    }
}
```

#### Using a List of Lists

```java
class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> tri = new ArrayList();
        List<Integer> firstList = new ArrayList();
        tri.add(firstList);
        firstList.add(1);

        for(int i = 1; i < numRows; i++ ){
            List<Integer> prev = tri.get(i - 1);
            List<Integer> curr = new ArrayList<Integer>();
            for(int j = 0; j <= i; j++){
                if(j == 0 || j == i){
                    curr.add(j, 1);
                } else {
                    curr.add(j, prev.get(j) + prev.get(j - 1));
                }
            }
            tri.add(i,curr);
        }
        return tri;
    }
}
```
