[Question](https://leetcode.com/problems/next-permutation/)
### Optimal Solution

[Video Explanation](https://www.youtube.com/watch?v=JDOXKqF60RQ)

The problem wants us to find the next permutations, that mean that if our current array is 

`[2,1,4,5,2,1]` -> 214521 the number right greater than this is 215124 -> `[2,1,5,1,2,4]`

So what we did was first was to

1. Iterate from the very last and find the breaking point (where the current elements is less than the previous one)
2. Then find the smallest element greater than the current variable from the breaking point to the end of the array.
3. Swap these two number.
4. Now the array after the breaking point would be in sorted order, reverse it. And there you go.

```java
class Solution {
    public void nextPermutation(int[] nums) {
        for(int i = nums.length - 2; i >= 0; i--){
            if(nums[i] < nums[i + 1]){
                int minPos = i + 1;

                for(int j = i; j < nums.length; j++){
                    if(nums[minPos] >= nums[j] && nums[j] > nums[i])
                        minPos = j;
                }

                swap(nums, minPos, i);
                reverse(nums, i + 1, nums.length - 1);
                return;
            }
        }
        reverse(nums, 0, nums.length - 1);
    }

    private static void reverse(int[] nums, int start, int end){
        while(start < end) {
            swap(nums, start, end);
            start++;
            end--;
        }
    }
    
    private static void swap(int[] nums, int start, int end){
        int temp = nums[start];
        nums[start] = nums[end];
        nums[end] = temp;
    }
}
```
