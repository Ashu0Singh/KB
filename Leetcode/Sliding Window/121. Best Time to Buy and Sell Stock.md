### Optimal Solution

So here, we need to be sure that the maximum falls after the minimum since you cannot sell first and buy later, so to fix that keep a min and max to calculate the difference, keep updating the maximum. And whenever the minimum is updated refresh the maximum value.

```java
class Solution {
    public int maxProfit(int[] prices) {
        int min = prices[0];
        int max = prices[0];
        int profit = 0;
        for(int num : nums){
            if(num < min) {
                min = num;
                max = num;
            } else if (num > max) {
                max = num;
            }
            profit = Math.max(profit, max - min);
        }
        return profit;
    }
}
```
