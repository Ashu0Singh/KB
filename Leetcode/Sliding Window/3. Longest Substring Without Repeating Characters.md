[Question](https://leetcode.com/problems/longest-substring-without-repeating-characters/)

### Optimal Solution

So first we created a map of 256 characters and then started a sliding window from start, the window size would increase and with that the included character count would go up. If the included character is already there in the window we will move the starting point of the window by one. And decrease the char count. And every time we will calculate the length of the window to make sure we have the max length recorded. 

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int[] map = new int[256];
        int start = 0, end = 0, max = 0;
        char[] str = s.toCharArray();
        while(start <= end && end < str.length){
            char chStart = str[start];
            char chEnd = str[end];
            if(map[chEnd] == 0) {
                map[chEnd]++;
                end++;
            }
            else {
                map[chStart]--;
                start++;
            }
            if(max < (end - start)) max = end - start;
        }
        return max;
    }
}
```
