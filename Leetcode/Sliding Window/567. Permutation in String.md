### Optimal Soultion 

So the idea is to create two maps and compare them with a static size moving sliding window. The comparison will be O(26) and you are gonna do it n time so the worst case complexity of this algorithm would O(n). 

```java
class Solution {
    public boolean checkInclusion(String s1, String s2) {
        int windowSize = s1.length();
        int[] s1Map = new int[26];
        for(char ch : s1.toCharArray()){
            s1Map[ch - 'a']++;
        }
        int[] s2Map = new int[26];
        int start = 0, end = 0;
        while(start <= end && end < s2.length()){
            if((end - start) == windowSize ){
                s2Map[s2.charAt(start) - 'a']--;
                start++;
            }
            s2Map[s2.charAt(end) - 'a']++;
            end++;
            if(compareMap(s1Map, s2Map)) return true;
        }
        return false;
    }

    static boolean compareMap(int[] s1Map, int[] s2Map){
        for(int i = 0; i < 26; i++){
            if(s1Map[i] != s2Map[i]) return false;
        }
        return true;
    }
}
```
