### Optimal Solution

So for every integer in the list we will have two options, either add them or don't add them. So what we're doing here is creating a shallow copy of all the list's present in the answer list and adding our current integer to it and push the copy list to the answers list. So now we have both the cases covered, one where we have added our current element and second where we've not added our current element. 

```java
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> answer = new ArrayList<>();
        answer.add(new ArrayList<>());

        for(int num : nums){
            int size = answer.size();
            for(int i = 0; i < size; i++){
                List<Integer> newList = new ArrayList<>(answer.get(i));
                newList.add(num);
                answer.add(newList);
            }
        }

        return answer;
    }
}
```
