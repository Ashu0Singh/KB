### Optimal Solution

So as the question wants use to not use any element twice. And we can have duplicates elements. So we're gonna run the same algo as [[39. Combination Sum]] but with slight modifications. 
1. Sort the array before you push it for the recursive calls.
2. If you the element and then move forward increase the index by one so that you don't add the same element again.
3. If you're not gonna add that element so make sure to skip all the occurence of that element, and since the array is already sorter it's gonna be as easy as running while loop. 

```java
class Solution {
    public List<List<Integer>> combinationSum2(int[] candidates, int target) {
        List<List<Integer>> subsets = new ArrayList<>();
        Arrays.sort(candidates);

        helper(new ArrayList<Integer>(), subsets, candidates, 0, target);

        return subsets;
    }
    private static void helper(List<Integer> subset, List<List<Integer>> subsets, int[] nums, int index, int target){
        if (target == 0){
            subsets.add(new ArrayList(subset));
        } else if ( target > 0 && index < nums.length){
            subset.add(nums[index]);
            helper(subset, subsets, nums, index + 1, target - nums[index]);

            subset.remove(subset.size() - 1);
            while(index < nums.length - 1 && nums[index] == nums[index + 1]) index++;

            helper(subset, subsets, nums, index + 1, target);
        }
    }
}
```
