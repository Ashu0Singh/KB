[Question](https://leetcode.com/problems/permutations/)
### Optimal Solution

Just refer to [[78. Subsets]] to understand the initial part of the solution, what we're doing here is taking all the element in the array and the inserting the current element at ever possible position we could in that array. This way we will get all the permutation of the current elements and all the elements in the array. 

The removal at the end is to remove all the subsets since the listSize is the old one we can remove all the old subsets and only have the recently added ones in the array. 

```java
class Solution {
    public List<List<Integer>> permute(int[] nums) {
        return helper(nums, 0);
    }

    private List<List<Integer>> helper(int[] nums, int index){

        if(index >= nums.length) {
            List<List<Integer>> lists = new ArrayList<>();
            lists.add(new ArrayList<>());
            return lists;
        }

        List<List<Integer>> lists = helper(nums, index + 1);
        int listSize = lists.size();

        int j = 0;
        while(j < listSize){
            List<Integer> list = lists.get(j);

            for(int i = 0; i <= list.size(); i++){
                List<Integer> temp = new ArrayList<>(list);
                temp.add(i,nums[index]);
                lists.add(temp);
            }

            j++;
        }

        while(listSize-- > 0){
            lists.remove(0);
        }
        return lists;
    }
}
```



```java
class Solution {
    public List<List<Integer>> permute(int[] nums) {
        return helper(nums, 0);
    }

    private List<List<Integer>> helper(int[] nums, int index){

        if(index >= nums.length) {
            List<List<Integer>> lists = new ArrayList<>();
            lists.add(new ArrayList<>());
            return lists;
        }

        List<List<Integer>> lists = helper(nums, index + 1);
        List<List<Integer>> tempList = new ArrayList<>();

        for(List<Integer> list : lists){
            for(int i = 0; i <= list.size(); i++){
                List<Integer> temp = new ArrayList<>(list);
                temp.add(i,nums[index]);
                tempList.add(temp);
            }
        }
        return tempList;
    }
}
```
