[Question](https://leetcode.com/problems/kth-largest-element-in-a-stream/)

### Optimal Solution 

```java
class KthLargest {
    Queue<Integer> scores;
    int k;
    public KthLargest(int k, int[] nums) {
        this.scores = new PriorityQueue<Integer>();
        this.k = k;

        for(int num : nums){
            scores.offer(num);

            if(scores.size() > k) {
                scores.poll();
            }
        }
        
    }
    
    public int add(int val) {
        scores.offer(val);

        if(scores.size() > k) {
            scores.poll();
        }

        return scores.peek();
    }
}
```
