## Notes
This is how you write a custom comparator for a priority queue

```java
PriorityQueue<int[]> pq = new PriorityQueue<>(
	(a, b) -> Integer.compare(
		b[0]*b[0] + b[1]*b[1],
		a[0]*a[0] + a[1]*a[1]
	)
);
```
