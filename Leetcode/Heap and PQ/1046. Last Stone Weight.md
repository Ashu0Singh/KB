[Question](https://leetcode.com/problems/last-stone-weight/)

### Optimal Solution

```java
class Solution {
    Queue<Integer> stoneWeight = new PriorityQueue<>(Collections.reverseOrder());
    public int lastStoneWeight(int[] stones) {
        for(int stone : stones){
            stoneWeight.offer(stone);
        }

        while(stoneWeight.size() > 1){
            int x = stoneWeight.poll();
            int y = stoneWeight.poll();

            if(x > y) stoneWeight.offer(x - y);
        }

        if(stoneWeight.isEmpty()) return 0;
        return stoneWeight.peek();
    }
}
```
