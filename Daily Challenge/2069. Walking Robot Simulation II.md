### Optimal Solution

Read [[874. Walking Robot Simulation]] first. So here since we're not being provided with a direction, we're just gonna add the steps and the realise it when we need the data. This will decrease the number of computations we're doing. The lazy step is the one taking care of realising the current status at the time of being fetched. So what we're doing here is that, since the area is a cyclic in nature, we will calculate the perimeter and then mod the number of steps moved with that. And this will give us how much are we gonna move. The below solutions beats 100% in runtime!

```java
class Robot {
    int xMax, yMax, perimeter;

    int x, y;
    int num;

    int[][] dir = {{0,1}, {1,0}, {0,-1}, {-1,0}};
    String[] nameDir = new String[]{"East", "North", "West", "South"};

    int d = 0;

    public Robot(int width, int height) {
        xMax = width - 1;
        yMax = height - 1;
        perimeter = 2 * (xMax + yMax);
        x = y = 0;
        num = 0;
    }

    public void lazyStep(){
        num %= perimeter;
        //To change the direction since if we're starting from 0,0 and complete a full cycle the direction would be South but it will be east.
        if(num == 0) num = perimeter;
        while (num > 0) {
            if (d == 0 && (x + num) > xMax) {
                int move = xMax - x;
                num -= move;
                x = xMax;
            } else if (d == 1 && (y + num) > yMax) {
                int move = yMax - y;
                num -= move;
                y = yMax;
            } else if (d == 2 && (x - num) < 0) {
                int move = x;
                num -= move;
                x = 0;
            } else if (d == 3 && (y - num) < 0) {
                int move = y;
                num -= move;
                y = 0;
            } else {
                x = x + (dir[d][1] * num);
                y = y + (dir[d][0] * num);
                break;
            }
            changeDir();
        }
        this.num = 0;
    }
    
    public void step(int num) {
        this.num += num;
    }

    private void changeDir(){
        d = (d + 1) % 4;
    }
    
    public int[] getPos() {
        if(num > 0) lazyStep();
        return new int[]{x , y};
    }
    
    public String getDir() {
        if(num > 0) lazyStep();
        return nameDir[d];
    }
}

 * String param_3 = obj.getDir();
 */
```
